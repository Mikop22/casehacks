# Scotiabank Design Kit

Public-reference design kit for a Scotiabank case competition or hackathon.

This is not an official Scotiabank brand guide. It is a practical, public-site-derived kit based on the live Canadian Scotiabank website inspected on 2026-05-23:

- https://www.scotiabank.com/ca/
- https://www.scotiabank.com/ca/en/personal.html

## Files

- `scotia-design-language.md` - design audit, UI guidance, slide deck guidance, and do/don't rules.
- `scotia-tokens.css` - CSS custom properties and starter component classes for software outputs.
- `scotia-slide-master.md` - slide master rules and a case competition deck outline.
- `asset-inventory.md` - downloaded public assets, source URLs, and usage cautions.
- `assets/scotiabank-public/` - small extracted asset set for quick prototyping.

## Practical Use

For software, use `scotia-tokens.css` as the base layer, keep the UI mostly white/light gray, reserve Scotia red for primary actions and key status, and avoid decorative gradients.

For slides, use a white canvas, red section markers, black/charcoal headlines, measured gray text, full-bleed public imagery only where it clarifies the story, and one point per slide.
